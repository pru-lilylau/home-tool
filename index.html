雲端同步設定

用你自己嘅 GitHub 帳戶，將資料加密備份到私人 Gist，可以跨裝置同步。喺第二部裝置，只要輸入返同一個 Token 同 PIN（Gist ID 留空），就會自動搵返同一份資料，唔使手動抄 Gist ID。資料喺上傳前會用你嘅 PIN 碼加密，Token 只會存喺呢部裝置度。

注意：請用 GitHub 嘅 classic personal access token（Settings → Developer settings → Personal access tokens → Tokens (classic)），並勾選 gist 權限。新式嘅 fine-grained token 唔支援 Gist，會連接失敗。
